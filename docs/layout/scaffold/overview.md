## 参数概述

| 参数 | 功能 |
| -------|------|
| **[bottomBar](bottombar.md)**| 设置屏幕底部的导航栏 |
| **[drawerContent](drawercontent.md)** | 创建一个侧边应用栏 |
| **[floatingActionButton](floatingactionbutton.md)** | 创建可以位于底部的按钮 |
| **[topBar](topbar.md)** | 设置屏幕顶部的应用栏 |
